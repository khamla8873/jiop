📝 ໂຄ້ດຫຍໍ້ (Arduino)

#include <RTClib.h>
RTC_DS3231 rtc;
int relay = 7;

void setup(){
  pinMode(relay, OUTPUT);
  rtc.begin();
}

void loop(){
  DateTime now = rtc.now();
  // ປ້ຳ 08:00 ແລະ 18:00
  if((now.hour()==8||now.hour()==18)
    && now.minute()==0){
    digitalWrite(relay, HIGH); // ON
    delay(60000); // 1 ນາທີ
    digitalWrite(relay, LOW); // OFF
  }
}

⚙️ ຈຸດເດັ່ນ NE555

  • ວົງຈອນງ່າຍ — ໃຊ້ IC ຕົວດຽວ
  • ລາຄາຖືກ ຫຼາຍກວ່າ Arduino
  • ຮຽນຮູ້ electronics ໂດຍກົງ
  • ປັບ R2/C1 ເພື່ອປ່ຽນໄລຍະເວລາ
  • ໄຟໟ້ານ້ອຍ — ເໝາະ Solar

⏱️ ສູດຄຳນວນເວລາ

🕐 T_on = 0.693 × (R1+R2) × C
🕑 T_off = 0.693 × R2 × C

ຕົວຢ່າງ: ປ້ຳ 1 ນາທີ ທຸກ 30 ນາທີ
R1=100K, R2=2.2M, C=100μF

🔧 ຂັ້ນຕອນການຕໍ່

1
Battery 12V+ → VCC rail, Battery− → GND rail
2
NE555 Pin8(VCC)←VCC, Pin1(GND)←GND
3
Pin6,7 ← R1(100KΩ) ← VCC; Pin2 ← R2(ປັບໄດ້) ← Pin6; C1(100μF) ← Pin2 → GND
4
Pin3(OUT) → 10KΩ VR → Gate IRF540N MOSFET
5
MOSFET Drain → Relay coil → VCC; Source → GND
6
Relay NO/COM → Motor 12V; Diode 1N4007 ຂວາງ Motor
